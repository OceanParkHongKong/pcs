## Development

### VS Code Setup
- In VS Code, select target "Chrome" from the bottom right corner

### Debug Configuration
- When developing in VS Code, in `utils.dart`, change port to 8000 for debug flutter

### Building for Production
```bash
flutter build web --release
```
